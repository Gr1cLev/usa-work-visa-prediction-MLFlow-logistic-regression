Simulasi Visa LCA

Aplikasi ini membantu Anda mensimulasikan hasil permohonan visa kerja (Labor Condition Application/LCA). Cukup pilih atau isi data utama, lalu model akan memperkirakan apakah permohonan cenderung disetujui atau ditolak.

Cara pakai singkat: tetapkan alamat API (misal http://localhost:8000), pilih pilihan yang paling cocok, gunakan opsi "Lainnya" bila butuh data khusus, kemudian tekan tombol Prediksi.

Isi Data Permohonan
Contoh: kontrak, musiman.
